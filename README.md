# 💰 Python Expense Tracker

## 📌 Project Description
This is a simple Python-based expense tracker that allows users to record daily expenses, view category-wise summaries, and visualize spending using a pie chart.

---

## 🚀 Features
- Add expenses by category (Food, Travel, etc.)
- Calculate total expenses
- Display category-wise breakdown
- Visualize data using matplotlib (pie chart)

---

## 🛠️ Technologies Used
- Python
- Matplotlib

---

## ▶️ How to Run

1. Install matplotlib:
   pip install matplotlib

2. Run the program:
   python expense_tracker.py

---

## 📊 Sample Output

- Total expense displayed
- Category-wise breakdown
- Pie chart visualization
